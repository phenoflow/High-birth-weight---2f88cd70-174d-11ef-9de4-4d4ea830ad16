# High-birth-weight

## Requirements

[Docker](https://docs.docker.com/install/) and [CWLTool](https://github.com/common-workflow-language/cwltool#install)

## Configuration

Specify input dataset path in [High-birth-weight-inputs.yml](High-birth-weight-inputs.yml).
